# Linux Security Audit – Ubuntu 22.04 LTS

## Übersicht

Dieses Projekt zeigt einen Security-Audit für Ubuntu Linux 22.04 LTS.  
Die Bewertung orientiert sich an CIS Linux Benchmarks und Best Practices.

## Ziele

- System-Härtung prüfen
- Sicherheitslücken erkennen
- Risikobasierte Bewertung
- Handlungsempfehlungen liefern

## Scope

- Ubuntu 22.04 Server / Desktop
- Benutzerkonten, sudoers
- SSH Konfiguration
- Systemdienste / Firewall
- Auditing & Logging
- Kernel & Package Security

## Audit-Kategorien

- Benutzer- und Gruppenverwaltung
- Passwort- und Authentifizierungsrichtlinien
- SSH-Härtung
- Service-Härtung
- Logging / Auditd
- Firewall / IPTables
- Paket- und Update-Management

## Bewertungskriterien

- Konform
- Nicht konform
- Nicht anwendbar
- Manuelle Prüfung erforderlich
