# Audit Scope (Linux)

## In-Scope

- Ubuntu 22.04 LTS Server/Desktop
- Lokale Benutzerkonten und Gruppen
- SSH / PAM / sudo-Konfiguration
- Firewall / IPTables
- Auditd & Syslog
- Installed Packages & Security Updates

## Out-of-Scope

- Cloud-Provider-spezifische Einstellungen
- Container-Umgebungen (z. B. Docker/K8s)
- Anwendungen von Drittanbietern

## Voraussetzungen

- Root-Zugriff oder sudo-Rechte
- Zugriff auf Systemkonfiguration
